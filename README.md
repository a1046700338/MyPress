# MyPress
基于[VuePress](https://www.vuepress.cn/)修改的个人笔记网站

## 使用
## theme-default
请克隆该分支:origin
```shell
git clone -b origin git@github.com:a1046700338/MyPress.git
```
## theme-reco
请克隆该分支:theme-reco
```shell
git clone -b theme-reco git@github.com:a1046700338/MyPress.git
```

随后
```shell
npm i
```

## 运行
```shell
// 项目调试
npm run dev
// or
// npm run blog:dev
// 项目打包
npm run build
// or
// npm run blog:build
```
